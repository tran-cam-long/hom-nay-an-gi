# hom-nay-an-gi

A simple app to auto-update available eat-out options in real time and calculate suggestions based on factors such as device location, weather, season, personal preferences, and prices.

Quick start (Yarn)

1. Install dependencies:

```bash
yarn install
```

2. Run the aggregator service:

```bash
yarn start
```

Frontend demo

Save the provided background image as `frontend/bg-cats.png` and open `frontend/index.html` in a browser to view the sample UI.

Notes

- The project uses a small Node prototype at `src/index.js` (Overpass + optional Foursquare).
- For production, avoid relying on public Overpass endpoints without caching or self-hosting.
